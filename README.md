# 蔡森技术分析工具 v2.0 (Cai Sen Technical Analysis Tool)

基于台湾知名投资名师 **蔡森** 的技术分析方法论开发的自动化股票分析工具。

## 🧠 蔡森核心方法论

蔡森, 26年实战经验, 从50万操作获利2800万(2011年欧债危机期间)。他只看 **量与价**，不用任何滞后指标(MACD/RSI/KD)。

### 核心理念

| 要素 | 说明 |
|------|------|
| **量价为王** | 量价是最当下、最即时的信号，其他指标都是滞后指标 |
| **型态学** | 通过K线型态识别主力动向：进货→拉抬→出货 |
| **严设止损** | 每笔交易必须设好止损点，看错立即出场 |
| **波段操作** | 以长线为基础(日线/周线)，追求大波段利润 |
| **小赔大赚** | 止损风险5%，预估获利至少20%以上(3:1风险回报比) |

### 两大核心操盘技巧

#### 1️⃣ 破底翻 (做多信号)
```
定义: 股价在底部盘整 → 跌破底部颈线 → 又翻回颈线之上
含义: 有人在护盘(主力甩轿后拉升)
操作: 翻回颈线时买入，止损设在破底处
蔡森金句: "破底翻大都会越过前高"
```

#### 2️⃣ 假突破 (做空信号)
```
定义: 股价突破前高/颈线 → 随后又跌回颈线之下
含义: 主力在诱多出货(骗线)
操作: 跌回颈线时做空，止损设在突破高点
高档假突破更可靠，配合量价背离确认
```

### 涨幅满足计算
```
蔡森公式: 底部到颈线的距离 = 突破颈线后的等幅距离
例: 底部18元, 颸线20元, 距离2元
    → 突破20元后, 目标1 = 20 + 2 = 22元
    → 目标2 = 20 + 2×1.618 = 23.24元 (黄金比例)
```

### v2.0 新增: 多时间框架分析
```
日线: 看趋势方向，识别型态
周线: "量先价行上档无压" — 周线破底翻信号更重大
5分钟/小时线: 找精确入场时机 (蔡森: 用短周期抢得先机)
```

### v2.0 新增: 其他型态
- **岛型反转**: 跳空缺口后形成价格岛屿，反向缺口封闭
- **量先价行**: 量能大幅放大但价格尚未突破 → 即将突破
- **支撑分级**: 短期强支撑 / 短期弱支撑 / 长线大支撑

### 蔡森三心法则
- **耐心** — 等待型态完成
- **决心** — 信号出现果断进场
- **平常心** — 设好止损，不焦虑短线震荡

### 蔡森操作提醒
- 破底翻大都会越过前高
- 利空时看收盘价判断支撑，无利空时盘中破就跑
- 量价是所有技术指标之首
- 不要做短线，波段操作才是成功之道

## 🛠 工具功能

1. **自动型态识别**: 破底翻、假突破、W底、头肩底/顶、岛型反转、量先价行
2. **多时间框架**: 日线 + 周线同步分析
3. **颈线自动检测**: 支撑线和压力线识别
4. **量价关系分析**: 量价背离、量先价行检测
5. **涨幅满足计算**: 双目标 (等幅距离 + 黄金比例)
6. **止损点建议**: 根据型态自动设置止损
7. **支撑分级**: 短期/长线支撑分类
8. **批量扫描**: 多只股票同时分析

## 🚀 使用方法

### 单股分析
```bash
python cai_sen_analyzer.py AAPL          # 苹果
python cai_sen_analyzer.py 2330.TW       # 台积电
python cai_sen_analyzer.py 600519.SS     # 贵州茅台
python cai_sen_analyzer.py 000001.SZ     # 平安银行
python cai_sen_analyzer.py 0700.HK       # 腾讯
```

### 批量扫描
```bash
python cai_sen_analyzer.py AAPL TSLA NVDA MSFT
```

### 生成图表
```bash
python chart.py AAPL 6mo
```
